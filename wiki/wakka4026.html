Форма “Company”


Форма для реквизитов вашей компании(й), которые в дальнейшем будут использоваться в выставляемых счетах операторам.



При необходимости, можно завести несколько своих компаний, с разными реквизитами и работать с разными операторами от лица разных компаний. Здесь вы также можете загрузить лого вашей компании, которое будет добавляться в инвойсы в формате html. Инвойсы в эксель формате будут строится без лого.
Для того чтобы указать компанию от лица которой будет вестись работа с оператором нужно в разделе операторов в параметрах оператора назначить ему нужную компанию.

Для управления формой параметров оператора внизу формы расположены следующие кнопки управления:

КнопкаНазначение
Add companyДобавляет новую компанию с реквизитами указанными в форме.

Update attributesОбновляет информацию о комнании.

Delete companyУдаляет выбранную компанию.

Set defaultНазначает выбранную компанию основной (по умолчанию). Основная компания будет автоматически назначаться всем новым операторам добавляемым в систему.


Примечание: В системе всегда должна быть заведена как минимум одна компания. Наличие компании будет прямо сказываться на функцию построения инвойсов для операторов.
