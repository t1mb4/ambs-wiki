Форма “Radius Online Calls”


Форма позволяет следить за активными звонками, управляемыми радиус-процессом.
На основании данных из этой формы работает параметр “capacity” для направлений оригинирующих и терминирующих операторов.
Параметр “capacity” указывается для направлений в тарифных планах операторов (Tariffs > Edit tariffs).
Внешний вид формы :



Форма раздела состоит из следующих полей:


ПолеНазначение
Select originate operatorВы можете выбрать конкретного оригинирующего оператора для просмотра только его звонков.

Select originate ipЕсли вы выбрали конкретного оператора, то можете выбрать у него конкретный ip-адрес для просмотра.

Select terminate operatorВы можете выбрать конкретного терминирующего оператора для просмотра только его звонков.

Select terminate ipЕсли вы выбрали конкретного оператора, то можете выбрать у него конкретный ip-адрес для просмотра.

Enter destination numberЕсли требуется получить лист звонков, начинающихся на определенные цифры кода, то в поле нужно ввести эти цифры.

Refresh time (sec.)Время обновления информации, как часто AMBS будет забирать данные с MVTS-сервера. Не рекомендуется ставить значение меньше 10 секунд, может привести к зависанию MVTS-сервера.

Data formatТип вывода на экран данных.
Может принимать следующие значения:
detail – Будет выдаваться полная детальная информация в виде списка звонков.
summary – Суммарная статистика в виде двух полей: количество нулевых звонков и ненулевых.
summary operator – Суммарная статистика по операторам. Сколько сейчас проходит звонков от каждого оператора.

Originated destinationsНазвание направления по оригинационному тарифу оператора

Source ipПоказывать адрес, откуда пришел звонок.

Originate operatorПоказывать оригинирующего оператора.

Originate priceЦена оригинации данного звонка.

Source numberНомер вызывающего абонента.

Terminate operatorПоказывать терминирующего оператора.

Destination ipПоказывать адрес получателя.

Terminate priceЦена терминации данного звонка.

Destination numberНомер вызываемого абонента.

Elapsed timeВремя, прошедшее с начала звонка.

CallidИдентификатор звонка.

ConfidИдентификатор конфигурации, которая использовалась для звонков.

Terminated destinationsНазвание направления по терминационному тарифу оператора.

Route retriesКоличество попыток маршрутизации звонка.


Внизу формы расположены кнопки управления:


КнопкаНазначение
Set parametersПрименяет заданные параметры и обновляет страницу раздела.

Clear callsСбрасывает статистику по звонкам и обновляет страницу раздела.

Save optionsСохраняет настройки параметров для пользователя, чтобы они использовались в данном разделе постоянно.
