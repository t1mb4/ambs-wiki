Форма “Managers”


Форма менеджеров компании предназначена для ведения списка менеджеров, которые занимаются работой с операторами.



Для каждого менеджера в форме можно указать следующие данные данными:
  • ФИО менеджера
  • E-mail менеджера
  • Контактный телефон

Кнопки управления формы:

ПолеНазначение
Add managerДля добавления нового менеджера.

Update infoСохраняет внесенные изменения в параметрах менеджера.

Delete managerУдаляет выбранного менеджера.
