Примеры добавления карт в лот


Для того чтобы добавить новую группу карточек в лот, следует выполнить следующие действия:

1. Выбрать лот в поле «Select lot»;
2. Открыть форму по созданию карточек, путем нажатия кнопки «Card Generation»;
3. Выбрать в поле «Cards generate» режим создания карточек: group;
4. В поле «Authorization type» выбрать тип авторизации создаваемых карт;
5. В поле «Amount» указать количество единиц на одной карте;
6. В поле «Service» указать тип создаваемых карт;
7. В поле «Cards count» указать количество создаваемых карт;
8. Указать префикс новых карточек (первые цифры пина) в поле «Pin prefix»;
9. В поле «Max. sessions» указать количество одновременных звонков;
10. В поле «Category» выбрать категорию карт;
11. Нажать кнопку «Apply change» чтобы добавить карты.




Для того что бы добавить новую карточку в состав лота, следует выполнить следующие действия:

1. Выбрать лот в поле «Select lot»;
2. Открыть форму по созданию карточек, путем нажатия кнопки «Card Generation»;
3. Выбрать в поле «Cards generate» режим создания карточек: single;
4. В поле «Authorization type» выбрать тип авторизации создаваемой карты;
5. В поле «Amount» указать количество единиц на этой карте;
6. В поле «Manual pin()» указать префикс создаваемой карты;
7. В поле «Manual key()» указать пин-ключ создаваемой карты;
8. В поле «Max. sessions» указать количество одновременных звонков создаваемой карты;
9. В поле «Category» выбрать категорию создаваемой карты;
10. Нажать кнопку «Apply change» чтобы добавить карты.
