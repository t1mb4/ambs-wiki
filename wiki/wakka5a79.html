Примеры настройки маршрутизации


Для того чтобы воспользоваться функциями данного раздела, предварительно нужно:


1. Выбрать сайт оригинирующего оператора в поле “Select site”, для которого вы будете производить настройку маршрутизации.


2. В поле “Select country” нужно выбрать страну, направление которой будет редактироваться или в поле “Filter code” указать конкретный код искомого направления.


3. Нажать кнопку «Поиска» () для поиска направления.


4. Для того чтобы приступить к настройке маршрутизации направления, его нужно предварительно разблокировать, поставив галочку в поле “enable” направления. Можно одновременно задавать условия маршрутизации для нескольких направлений.


5. Задать необходимые параметры маршрутизации.


6. Поставить галочку в поле “Route parameters enable”.


7. Нажать кнопку “Update route parameters” для принятия внесенных изменений.


Чтобы удалить заданные ранее параметры ручной настройки маршрутизации направлений, нужно:


1. Найти направление на которое были установлены правила маршрутизации


2. Активировать направление, поставить галочку в первое поле строки направления.


3. Нажать кнопку “Delete from enabled list”().


Параметры настройки маршрутизации для всех выбранных направлений будут удалены.
