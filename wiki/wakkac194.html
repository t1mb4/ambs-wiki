Statistics by destinations. Сводная статистика работы


Форма предназначена для просмотра суммарной статистики по прошедшим звонкам, по направлениям, за выбранный период времени.
Внешний вид формы представлен на следующем рисунке:



Описание полей формы:


ПолеНазначение
Select operatorПозволяет выбрать для просмотра конкретного оператора, по которому необходима статистика звонков по направлениям.

Select server MeraПозволяет выводить статистику прошедших звонков по направлениям через конкретный адрес MTVS-сервера.
Значение по умолчанию – All (все MTVS-серверы).

Destination typeТип статистики по направлениям. Может принимать следующие значения:
Terminate – на дисплей ПК будет выводиться статистика по прошедшим звонкам, только по терминационным направлениям;
Originate – на дисплей ПК будет выводиться статистика по прошедшим звонкам, только по оригинационным направлениям;
All – на дисплей ПК будет выводиться статистика по прошедшим звонкам, по оригинационным и терминационным направлениям.
Значение по умолчанию – All.

Time periodПериод времени, за который будут показаны звонки.
Может принимать следующие значения:
Today – сегодня;
Yesterday – вчера;
3 days – три дня;
7 days – семь дней, не включая текущий день;
Month – текущий месяц;
Last month – предыдущий месяц;
Period – период, заданный вручную полями From-To.
Значение по умолчанию: Today.

Period fromДата начала периода, за который будет выбираться статистика – задается вручную.

Period toДата окончания периода, за который будет выбираться статистика – задается вручную.

Last 24 hourДополнительный фильтр, позволяющий выставлять временной интервал в часах за определенный период времени.

First numbers of destinationПозволяет выводить статистику по прошедшим звонкам по конкретному направлению, код которого задается в ручную.

Graph typeГрафическое отображение статистики по прошедшим звонкам. График может строиться по следующим значениям:
Calls count – на графике будет показана диаграмма успешных звонков по отношению к общему количеству звонков за определенный период времени;
Minutes – на графике будет показана диаграмма средней продолжительности звонков в минутах за определенный период времени;


В форме присутствуют следующие кнопки управления:


КнопкаНазначение
Show statПоказать статистику по прошедшим звонкам.

Show graphПоказать статистику по прошедшим звонкам в виде графического отображения.


В таблице статистики звонков по направлениям отображается следующая сводная информация:


ПолеНазначение
DestinationНазвание направления.

All MinutesОбщее количество минут.

All callsОбщее количество звонков.

Success callsКоличество успешных звонков, у которых код возврата – 16, 17 и 19.

Short callsПроцент коротких звонков, у которых продолжительность меньше 30 секунд.

CostСтоимость общего количества минут.

ASRПроцент успешных дозвонов.

Complete CallsПроцент ненулевых звонков.

ACDСредняя продолжительность разговора в минутах.

PercentПроцент нулевых звонков.

SummaryОбщий результат сводной информации по всем направлениям, на которые проходили звонки.
