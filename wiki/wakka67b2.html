Пример настройки маршрутизации


При настройке маршрутизации, используя функции формы «Operator route param», необходимо выполнить следующее:

1. Выбрать сайт оригинирующего оператора в поле «Select site», для которого дилер будет производить настройку маршрутизации;

2. В поле «Select country» следует выбрать страну, направление которой будет редактироваться или в поле «Filter code» указать конкретный код искомого направления;

3. Нажать кнопку «Фильтр» для поиска направления;

4. Приступая к настройке маршрутизации направления, дилеру необходимо предварительно разблокировать его, поставив галочку в поле «enable» направления. В качестве альтернативы, программа позволяет одновременно задавать условия маршрутизации для нескольких направлений;

5. Задать необходимые параметры маршрутизации;

6. Поставить галочку в поле «Route parameters enable»;

7. Нажать кнопку «Update route parameters» для принятия внесенных изменений.

Примечание:
При удалении заданных ранее параметров ручной настройки маршрутизации направлений, следует:
  • Найти направление, на которое были установлены правила маршрутизации;
  • Активировать направление, поставив галочку в поле «enable» строки направления;
  • Нажать кнопку «Delete from enabled» list . Параметры настройки маршрутизации для всех выбранных направлений будут удалены.
