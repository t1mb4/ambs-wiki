Форма “Country Codes”


Раздел для ведения кодов и названий всех стран, заведенных в системе



Верхняя часть формы служит для добавления новых стран в базу системы.


ПолеНазначение
CountryНазвание страны.

Country codeКод страны.

Num countФильтр количества знаков в номере страны. Фильтра точного количества знаков в номерах страны.
Значение по умолчанию: 0
Пример: Россия – 11 знаков.
Num count – 11

Top range border

Фильтр диапазона допустимого количества знаков в номерах страны.
В этом поле указывается верхняя граница допустимого количества знаков в номерах.
При использовании этого поля, поле “Num count” будет выступать в роли нижней границы.
Значение по умолчанию: 0


Пример: Китай 10–13 знаков.
Num count – 11
Top range border – 13

Для того чтобы добавить новую страну достаточно заполнить поля с названием страны (Country name) и кодом страны (Country code).
Для того чтобы закончить операцию добавления новой страны, нужно нажать кнопку “Add new”.
Под формой расположены кнопки со стрелками для навигации по списку уже добавленных в систему стран.


Ниже отображается список со странами. Если необходимо произвести какие-нибудь действия с одной из стран, например, изменить ее код или длину фильтра номеров, предварительно нужно:


1. Найти страну в списке стран.

2. Выбрать эту страну (поставить точку в начале записи).

3. Внести необходимые изменения в форме.

4. Нажать на кнопку “Save changes” для сохранения изменений.

Для удаления одной из стран, нужно найти эту страну в списке, активировать ее запись, поставив галочку в поле “chk” и нажать на кнопку “Delete”.


Замечание: При удалении страны, автоматически станут недоступными все направления, в которых использовался код этой страны. Поэтому, чтобы не оставлять недействительных записей в базе, перед удалением страны рекомендуется удалить все ее направления в разделе “Edit destination”.
