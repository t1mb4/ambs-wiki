Detail statistics. Детальная статистика работы


Форма предназначена для просмотра детальной статистики по прошедшим звонкам за выбранный период времени.
Внешний вид формы показан на следующем рисунке:



Левая часть формы служит для выбора доступных операторов и установки фильтров для получения детальной статистики.
Правая часть позволяет выбрать период времени, за который будете показана статистика и тип статистики.


Описание полей формы:


ПолеНазначение
Select operatorПозволяет выбрать для просмотра доступного оператора по нескольким параметрам:
Originate – просмотр доступного оператора только по оригинации;
Terminate – просмотр доступного оператора только по терминации;
All ip – просмотр статистики по звонкам от доступного оператора по всем или по конкретному MTVS-серверам. Значение по умолчанию – all ip;
All sites – просмотр статистики звонков по всем или по отдельно взятому сайтам доступного оператора. Значение по умолчанию – all sites.

Destination codeФильтр по номеру. Если требуется получить список звонков, начинающихся на определенные цифры номера, их следует ввести в этом поле.

Source numberФильтр по шлюзу. Адрес исходящего шлюза, с которого поступает звонок.

Used currencyТип валюты, которая будет использоваться для вывода результатов.

Records countКоличество строк, которые будут показаны на дисплее ПК за один раз. Может принимать следующие значения: 10, 20, 30, 50, 100 или All (все).

ShowТип звонков, показываемых на дисплее ПК.
Может принимать следующие значения:
All – показать все звонки;
Not zero calls – показать только ненулевые звонки;
Zero calls only – показать только нулевые звонки;
Not calculated calls – показать не тарифицированные звонки. Звонки, для которых система не может найти тарифные планы для их тарификации;
Not calculated calls (term) – все звонки, не тарифицированные оригинирующим оператором;
Not calculated calls (orig) – все звонки, не тарифицированные терминирующим оператором;
Значение по умолчанию – All.

Output toФильтр отображения. Куда будет выведен результат.
Может принимать следующие значения:
Display – вывести на дисплей ПК;
File – записать в файл. Система сформирует файл формата .scv, совместимого с программой «Excel».
Значение по умолчанию: display

Time periodПериод времени, за который будут показаны звонки.
Может принимать следующие значения:
Today – сегодня;
Yesterday – вчера;
3 days – три дня;
7 days – семь дней, не включая текущий день;
Month – текущий месяц;
Last month – предыдущий месяц;
Period – период, заданный вручную полями From-To.
Значение по умолчанию – Today.

Period fromДата начала периода, за который будет выбираться статистика – задается вручную.

Period toДата окончания периода, за который будет выбираться статистика – задается вручную.

Select time from-toЧасовой интервал. Если необходимо просмотреть статистику за конкретный интервал времени в течение дня, например, с 14:00 до 18:00, в этих полях следует указать период, за который будет выводиться статистика.

Time difference (hours)Сдвиг часового пояса в часах. Используется при сверке статистик звонков с другими операторами.
Значение по умолчанию: 0 часов.


В форме присутствуют следующие кнопки управления:


КнопкаНазначение
Show callsПоказать статистику по звонкам с учетом заданных в форме раздела параметров.

Parameters listФорма полей отображения. Открывает окно с полями, которые будут показаны в статистике для каждого звонка. Поля соответствуют полям логов MVTS-сервера.


При просмотре детальной статистики, отображаемые поля с параметрами звонков можно регулировать в секции «Operators parameters», которая вызывается кнопкой «Parameters list»:


ПолеНазначение
IpАдрес шлюза оператора.

UserПользователь, зарегистрированный на MVTS-сервере.

Bytes inКоличество байт, принятых от оператора.

Bytes outКоличество байт, отправленных оператору.

Number inВходящий номер, с которого был отправлен звонок на входе.

Number billВходящий номер, с которого был отправлен звонок.

DST_NAMEНазвание направления, через которое проходят звонки.

CostСтоимость звонка.

PriceЦена единицы времени звонка.

Client_priceЦена единицы времени звонка оператора, доступного для просмотра.

Client_costСтоимость звонка по цене оператора, доступного для просмотра.

Connection timeВремя начала звонка.

Disconnection timeВремя окончания звонка.

Session timeПродолжительность сессии звонка.

Setup timeВремя установки соединения.

Bill dateДата прохождения звонка.

Bill timeВремя прохождения звонка.

Local timeЛокальное время оператора, доступного для просмотра. Если будет задан параметр «Time difference», время в этом поле будет отображаться с учетом заданной разницы во времени.

ConfidИдентификатор конфигурации, которая использовалась для звонков.

CallidИдентификатор звонка.

Disconnect codeКод MVTS-сервера при завершении звонка, показывает сторону, инициализировавшую разрыв.

Q931 codeКод, генерируемый MVTS-сервером при завершении звонка. Детальнее коды описаны в «Приложении А» данного руководства.

CodecКодек, использованный при прохождении звонка (входящий и исходящий).

Route retriesКоличество попыток маршрутизации звонка.

QOSПараметр качества сервиса. Берется с MVTS-сервера.

Lar fault reasonПоле MVTS-сервера, коды причин отказа.

ПолеНазначение
All ringsОбщее количество звонков по направлениям.

Success ringsКоличество успешных звонков, у которых код возврата – 16, 17 и 19.

ASRПроцент успешных дозвонов.

ACDСредняя продолжительность разговора в минутах.

Short callsПроцент коротких звонков, у которых продолжительность меньше 30 секунд.

All minutesОбщее количество минут.

Clients sumСумма денежных средств на балансе оператора.

SUM Terminate (Originate) (у.е.)Сумма терминации (оригинации) за прошедший объем трафика.
