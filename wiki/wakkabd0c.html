Форма “Active calls”


Форма по активным звонкам с карточек позволяет отследить все активные на данный момент времени сессии звонков, проходящие через карточную платформу. Используя эту форму, пользователь также может сбросить любую из этих сессий при необходимости.
Внешний вид формы представлен на следующем рисунке:



Описание полей формы:


ПолеНазначение
TypeТип активных звонков. Может принимать значения: VoIP или Dial-up (при условии, что установлен диал-ап модуль IP Studio AMBS).

Enter session IDПозволяет сбросить идентификационный номер сессии, под которой звонок проходит в системе.

Session IDИдентификационный номер сессии, под которой звонок проходит в системе. С помощью этого номера пользователь может сбросить сессию.

Conf IDИдентификатор звонка, под которым он проходит через оборудование карточной платформы.

Start timeВремя начала сессии разговора.

ANIНомер звонящего абонента.

PINПин-код, который использовался для звонка.

Card numberНомер карты, через которую проходит звонок.

DestinationНомер вызываемого абонента.
