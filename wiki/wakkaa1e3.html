Форма “ANI Reports”


Форма предназначена для просмотра детальной и суммарной статистики по АОН-номерам за выбранный период времени.
Внешний вид формы показан на следующем рисунке:



Верхняя часть формы служит для выбора АОН-операторов и установки фильтров, для получения детальной и суммарной статистики.
Нижняя часть позволяет выбрать период времени, за который будет показана статистика.


Описание полей формы:


ПолеНазначение
Select operator Позволяет выбрать для просмотра конкретного оператора, по которому необходима статистика на АОН-номера.
Значение по умолчанию: Any (любые операторы).

Build report for Вид сортировки, по которой будет выводиться статистика.
Может принимать следующие значения:
Site – показать отчет по сайту оператора;
ANI Group – показать отчет по АОН-группе, включающей в себя все АОН-сеты;
ANI Set – показать отчет по отдельно взятому сету.
Информацию по АОН-группам и АОН-сетам можно найти в разделе «Операторы» – Секция АОН-групп оператора.

Select Site or ANI set Дополнительный вид сортировки, для более детального вывода статистики.
Может принимать следующие значения:
Select site – показать статистику по всем сайтам или по отдельно взятому сайту оператора, если в предыдущей сортировке (см. выше) стоит параметр «Site»;
Select ANI Group – показать статистику по АОН-номерам по всем АОН-группам или по отдельно взятой группе, если в предыдущей сортировке (см. выше) стоит параметр «ANI Group»;
Select ANI set – показать статистику по АОН-номерам по всем АОН-сетам или по отдельному сету, если в предыдущей сортировке (см. выше) стоит параметр «ANI set».

Search by ANIПоиск статистики по отдельно взятому АОН-номеру.
Работает только при просмотре детальной статистики.

Statistic ModeТип статистики.
Может принимать следующие значения:
Summary – суммарная статистика, краткий вариант. Формируется быстро. Если стоит сортировка по ANI Group, то в этом случае появляется дополнительный параметр Grouped, который принимает два значения:
True– показать статистику суммарно по АОН-группе;
False – показать статистику суммарно по АОН-сетам в составе одной АОН-группы.
Значение по умолчанию: False.
Details – Детальная статистика, с полными выкладками по каждому звонку. Загружается дольше, чем суммарная.
Значение по умолчанию: Summary.

Show report by Дополнительный вид сортировки, по которому будет выводиться статистика.
Может принимать следующие значения:
Country – показать статистику по странам;
Dest. Operator – показать статистику по направлениям операторов.

Time periodПериод времени, за который будут показаны звонки.
Может принимать следующие значения:
Today – сегодня;
Yesterday – вчера;
3 days – три дня;
7 days – семь дней, не включая текущий день;
Month – текущий месяц;
Last month – предыдущий месяц;
Period – период, заданный вручную полями From-To.
Значение по умолчанию: Today.

Period fromДата начала периода, за который будет выбираться статистика – задается вручную.

Period toДата окончания периода, за который будет выбираться статистика – задается вручную.

Last 24 hour

Дополнительный фильтр, позволяющий выставлять временной интервал в часах за определенный период времени.


В форме присутствуют следующие кнопки управления:


КнопкаНазначение
Show CDR parametersФорма полей отображения. Открывает окно с полями, которые будут показаны в статистике, для каждого звонка. Поля соответствуют полям логов MVTS-сервера.

FilterОсуществляет поиск статистики по отдельно взятому АОН-номеру, работает в связке с полем Search by ANI.
