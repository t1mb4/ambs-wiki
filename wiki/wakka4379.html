Форма “Logs”


Форма логирования действий производимых пользователями в системе “IP Studio AMBS”.
Система автоматически логирует все основные действия, такие как работа с тарифными планами, работа с настройками системы и параметрами пользователей.
Внешний вид формы:



Описание полей формы:


ПолеНазначение
Select userВыпадающий список для выбора пользователя.
Если пользователь выбран, будет показана история действий только этого выбранного пользователя.

Select formВыпадающий список для выбора формы.
Если форма выбрана, будет показана история действий только в этой выбранной форме.

Select actionВыпадающий список для выбора действия.
Если действие выбранно, будет показана история только с этим, выбранным действием.

Time periodПериод времени, за который будут показаны звонки.
Может принимать следующие значения:
Today – Сегодня.
Yesterday – Вчера.
3 days – Три дня.
7 days – Семь дней, не включая текущий день.
Month – Текущий месяц.
Last month – Предыдущий месяц.
Period – Период, заданный вручную полями From-To.
Last 24 hours – Период за последние 24 часа, указывается в часах
Значение по умолчанию – today

Logs rows countКоличество строк истории действия показываемых за один раз на экран.

Search in logsПоиск указанного фрагмента текста в логах.

Запуск процесса выборки истории действий.


Отчет по действиям будет показан в следующем виде:



ПолеНазначение
User nameИмя пользователя под которым производилось действие.

Date-timeДата и время совершения действия.

FormФорма в которой производилось действие.

ActionТип производимого действия.

Log detailsДетальное описание действия.

FileФайл с загружаемой информацией.
Добавляется если производилась загрузка данных в систему (например загрузка тарифного плана из файла).
