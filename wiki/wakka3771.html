Форма “Mail Invoices”


Форма позволяет вам получить на выбранный адрес электронной почты инвойсы для указанных компаний, за выбранный расчетный период времени.
Рассылать инвойсы из этой формы нужно только на адреса сотрудников компании. В каждое письмо с инвойсом добавляется отчет по оператору с данными по наработкам из разделов “Call Details” и “New Statistic” за этот период времени, это позволит выявить любые проблемы и неточности в работе оператора и в самой статистике (если в этот расчетный период были сбои в работе системы.



Визуально, форма разделена на две части, в левой части вы можете выбирать операторов, для которых нужно подготовить инвойс:

  • Selected operators – для выборочной отправки
  • All operators – для всех операторов.

В правой части выбирается временной период, за который будут строиться инвойсы и адрес почты, куда эти инвойсы будут отправлены.
Адреса, на которые будут отправляться инвойсы можно редактировать в разделе “Managers” (Company -> Managers).
Функция “Group by” сгруппирует в инвойсе все направления с одинаковыми названиями и ценой.


По умолчанию все инвойсы будут создаваться в виде html-страниц заархивированных архивом “tar”, при желании этот тип можно сменить и получать счета в виде xls-страниц (MS Excel).
Для того чтобы сменить тип формата инвойсов нужно в конфигураторе системы (Administration – Configuration) изменить значение параметра “INV_SEND_FMT” на “xls” (значение по умолчанию – tar).
