Форма “Post Office”


Форма по рассылки сообщений операторам:



С помощью данной формы можно рассылать уведомления и любые другие сообщения операторам, которые заведены в системе. Письма будут рассылаться индивидуально, вне зависимости от списка выбранных операторов.

Визуально форму можно разделить на две части: левая часть отведена для деталей письма, в правой части находится список операторов, заведенных в системе, которым вы можете отправить письмо. Контактные адреса операторов берутся из данных оператора указанных в разделе “Properties” (Operators -> Edit operator -> Properties).

В форме присутствуют следующие элементы управления:

ПолеНазначение
FromАдрес пользователя, от имени которого будет отправлено письмо. Список пользователей берется из раздела “Managers” системы (Company -> Managers).

HeaderЗаголовок письма.

Mail bodyОсновной текст письма.

AttachmentВложения. Во вложениях к письму, система позволяет закрепить файл(ы).

SendОтправка письма.Письмо будет отправлено всем выбранным операторам.

Select mailТип электронного адреса, на который будет отправлено письмо:
Финансовый (Finance), письмо будет отправлено по адресу указанному в финансовых контактных данных.
Технический (Technical), письмо будет отправлено по адресу указанному в технических контактных данных оператора.
Оба (Both), письмо будет отправлено на финансовый и технический контакты.

List operatorsСписок операторов заведенных в системе, которым можно отправлять письма на их контактные адреса, которые берутся из данных этих оператора указанных в разделе “Properties” (Operators -> Edit operator -> Properties).

Show mail historyОткрывает форму логов работы с PostOffice-ом.
