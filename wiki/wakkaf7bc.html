Общие сведения и организация IP Studio AMBS. Иерархия системы


Данная система построена на базе использовании браузеров FireFox и Internet Explorer. Изначально, система разрабатывалась для транзитных операторов, но впоследствии по многочисленным заявкам наших клиентов были добавлены дополнительные модули, например, как работа с карточными платформами


Над системой постоянно ведется работа по исправлению неточностей и дополнению функциональных возможностей. На рисунке представлен общий вид системы:



Основное меню системы состоит из следующих разделов:

РазделОписание
Calls infoВ этом разделе можно просмотреть детальную информацию по звонкам и по операторам за любой период времен. Получить статистику звонков проходящих в текущий момент времени, протестировать варианты доступных направлений по конкретным номерам для определенных операторов и сравнить CDR-файлы.

TariffsРаздел создан для ведения работ с тарифными планами операторов: просмотр, создание и редактирование планов. Форма “List tariffs” позволяет просмотреть уже существующие тарифы, выставить “alert”-ы на необходимые направления по параметрам соединения (ASR или ACD) и сгенерировать прайс-листы из списка всех предоставляемых направлений или только из списка рабочих (за выбранный период времени). Цветовая раскраска позволяет быстро проанализировать работу направлений в удобной форме.

DestinationsРаздел для ведения стран и рабочих направлений. Просмотр, создание и редактирование кодов стран, регионов и городов. Добавление, редактирование и удаление направлений. Установка фильтров на определенное количество знаков в номере по конкретным направлениям. Также, форма “Trouble tickets” позволяет наглядно отслеживать статистику изменений параметров качества (ACD и ASR) по всем рабочим направлениям.

OperatorsРаздел позволяет добавлять новых операторов и редактировать их параметры. Настраивать ip-адреса операторов, закреплять за ними тарифы. Также здесь можно просмотреть полную статистику по операторам их ip-адресам и настройкам.

PaymentsРаздел для работы с балансом клиентов. Позволяет вести денежный учет с клиентами и в наглядной форме получить информацию о денежных движениях по любому оператору. Здесь же можно вести все денежные расчеты с карточными дилерами.

CompanyЗдесь можно указать, просмотреть и отредактировать информацию о своей собственной компании. Задать реквизиты компании, добавить агентов и менеджеров. Эта информация будет использоваться системой для выставления инвойсов и начисления процентов агентам.

EquipmentsРаздел создан для работы с MVTS серверами (просмотр, создание, обновление и редактирование параметров MVTS серверов), установки новых обновлений AMBS системы и работы с маршрутизаторами Cisco.

ReportsЗдесь можно просмотреть общую статистику по операторам за любой период времени, пересчитать звонки за прошлые числа, если возникли расхождения в статистике по форс-мажорным обстоятельствам и подготовить счета (инвойсы). Также в разделе можно получить детальную статистику по каждому оператору куда, сколько и по каким направлениям он звонил.

AdministrationРаздел для администрации AMBS системы. Здесь можно заводить и редактировать группы пользователей, настройки и права самих пользователей, а также заполнять и настраивать языковые словари системы.

PartnersРаздел для работы партнеров. Ваши партнеры могут использовать этот раздел для получения различных типов статистики по их деятельности с Вами и ведения их дальнейших тарифов на базе тарифов предоставляемых Вами, используя средства AMBS.

CardsРаздел для работы с карточными платформами. Здесь можно работать с карточными лотами (создавать, редактировать) и получать всевозможные типы статистики по работе самих карточек.
