Форма “Rate Groups”


Раздел для организации групп тарифных планов с их привязкой к номерам доступа (DNIS) или префиксам АОН-ов (ANI prefix). В процессе эксплуатации системы могут возникать ситуации, когда необходимо использовать не один тарифный план, а несколько, в зависимости от того, с какого номера пришел звонок.

Например, один карточный лот используется в нескольких городах, регионах. Звонки приходят на карточную платформу по разным номерам доступа и звонки из каждого города нужно тарифицировать по тарифным планам этих городов. Для того чтобы такая схема работала, нужно назначить карточному лоту группу тарифных планов, распределенных по номерам доступа(DNIS). Форма имеет следующий вид:



Поля формы:

ПолеНазначение
New rate groupПоле для ввода имени новой группы тарифных планов или для изменения имени уже существующей группы.

Exist rate groupsВыпадающий список с группами, уже добавленными в систему.

RatesСписок всех тарифных планов, заведенных в системе.
Служит для добавления тарифных планов в тарифную группу.
При добавлении выбранный тарифный план будет привязан к выбранному номеру доступа, или префиксу АОН-а.

DNISСписок всех номеров доступа заведенных в системе.
Служит для добавления номеров доступа в тарифную группу.
При добавлении выбранный номер доступа будет привязан к выбранному тарифному плану.

ANI prefixСписок всех префиксов АОН-ов заведенных в системе.
Служит для добавления префиксов АОН-ов в тарифную группу.
При добавлении выбранный префикс АОН-а будет привязан к выбранному тарифному плану.


Кнопки управления формы:

КнопкаНазначение
ShowПоказывает записи тарифной группы, выбранной из списка “Existing rate groups”.

Add rate groupСоздает в системе новую тарифную группу с именем, указанным в поле “New rate group”.

Change nameИзменяет имя тарифной группы, выбранной в списке “Existing rate groups” на новое имя, указанное в поле “New rate name”.

Add rate to planДобавляет выбранный в поле “Rates” тарифный план в текущую тарифную группу. Тарифный план будет привязан к номеру доступа, выбранному в поле “DNIS”.

Add DNISЕсли в системе отсутствует требуемый номер доступа или префикс АОН-а, их можно добавить в разделе “Edit destinations”.
Эта ссылка открывает в новом окне браузера раздел “Edit destinations” для добавления номера доступа.


Замечание: Следует помнить, что в одной тарифной группе одному номеру доступа(DNIS) или префиксу АОН-а может быть назначен только один тарифный план.
