Форма “Lot prefixes”


Форма предназначена для назначения лотам префиксов и входящих трансляций для идентификации лота в биллинге.



Описание полей формы:

ПолеНазначение
Select lotПозволяет выбрать из выпадающего списка необходимый лот.

Default prefixПозволяет установить префикс и входящую трансляцию для всех лотов по умолчанию.

PrefixПозволяет установить префикс за определенным лотом или префикс по умолчанию.

In_dst_translateПоле трансляции входящего номера с задаваемого адреса оператора.
Используется только для идентификации лота в биллинг.


В форме присутствуют следующие кнопки управления:

КнопкаНазначение
Set default prefixСохраняет внесенные изменения для поля «Default prefix».

AddСохраняет внесенные изменения для выбранного лота.

DeleteУдаляет внесенные изменения в выбранный лот.
