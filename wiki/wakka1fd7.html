Форма “Lot statistic”


Форма предназначена для просмотра статистики по прошедшим звонкам выбранного лота за выбранный период времени.
Внешний вид формы представлен на следующем рисунке:



Визуально, форму раздела можно разделить на две части. В левой части формы, вы можете выбирать лот, устанавливать фильтры на нулевые номера, выбирать тип показываемой статистики и количество строк показываемых на экране за один раз. Правая часть позволяет выбрать период времени, за который будете показана статистика.


Описание полей формы:


ПолеНазначение
Select lotПозволяет выбрать из выпадающего списка необходимый лот. Значение по умолчанию: Select lot.

Rows countКоличество выводимых записей на дисплей ПК. Может принимать следующие значения: 50, 100, 200, 300 и 500.
Значение по умолчанию: 50.

ShowПозволяет установить необходимые параметры для вывода статистики по прошедшим звонкам выбранного лота. Может принимать параметры:
summary/details – будет выводиться общая статистика или детальная по прошедшим звонкам лота;
VoIP/Dial-up/Pays history – будет показана статистика только по VoIP-карточкам в составе лота или по Dial-up-карточкам или показать историю платежей.

Show calls duration inПоказать продолжительность звонка. Может принимать параметры:
Seconds – продолжительность звонка будет выводиться в секундах;
hour:min:sec – продолжительность звонка будет выводиться по времени.

Not zero callsВ статистике будут показаны только не нулевые звонки.

NumberПоиск по номеру вызываемого абонента.

Time periodПериод времени, за который будут показаны прошедшие звонки по лоту.
Может принимать следующие значения:
Today – сегодня;
Yesterday – вчера;
3 days – три дня;
7 days – семь дней, не включая текущий день;
Month – текущий месяц;
Last month – предыдущий месяц;
Period – период, заданный вручную полями From-To.
Значение по умолчанию – Today.


В форме присутствуют следующие кнопки управления:


КнопкаНазначение
Show info. Выводит статистику по установленным параметрам.

Excel version. Позволяет сохранить данные по звонкам в файл формата .csv.


Вывод статистики звонков производится нажатием кнопки «Show info». Результат будет показан в виде таблицы с параметрами прошедших звонков:



При просмотре суммарной статистики, в таблице показывается следующая информация:


ПолеНазначение
LotНазвание карточного лота.

Calls countКоличество прошедших звонков.

Calls durationОбщая продолжительность разговоров.

CostОбщая стоимость разговоров.

SummaryИтоговая строка.


При просмотре детального списка звонков для каждого звонка будут показаны следующие поля:


ПолеНазначение
LotНазвание карточного лота.

Card numberНазвание карты, через которую проходил звонок.

Call timeВремя звонка.

DNISИспользованный номер доступа к карточной платформе.

ANIНомер звонящего абонента.

Dial numberНомер вызываемого абонента.

RateЦена направления, по которому прошел звонок.

Call durationПродолжительность звонка.

Rounded durationОкругленная продолжительность звонка.

Call costСтоимость прошедшего звонка.

Return codeКод возврата (код завершения) звонка.

Personal cardпоказывает закреплена ли карточки за каким-либо абонентом.


При большом списке прошедших звонков, детальная статистика может быть показана на нескольких страницах. Передвигаться между страницами статистики следует с помощью кнопок со стрелками, расположенных над таблицей со статистикой по звонкам.
