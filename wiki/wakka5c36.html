Форма “Dictionary”


Форма внутренних словарей системы. Под словарем понимается список всех текстовые надписей в системе. По умолчанию, с системой поставляется: базовый словарь (base) и русский (russian).
В этой форме можно создать другие словари и перевести все базовые надписи системы на тот язык, на котором будет удобнее работать с системой. Словари можно экспортировать и импортировать из стандартных CSV-файлов.



Форма словаря состоит из следующих полей:


ПолеОписание
Select langВыпадающее меню, для выбора подключенных к системе словарей.

Or enter newПоле, для имени нового словаря или изменения выбранного словаря.

Record countКоличество записей словаря показываемых на экран за один раз.

Show from idПоле идентификатора записи, с которого будет осуществляться вывод на экран.

Find phraseПоле для поиска фраз в словаре. Поиск происходит по точному совпадению фразы.
Для поиска по неполной фразе необходимо использовать символ – "%", означающий любое продолжение.
Например, фильтр для фразы, в которых встречается слово “price”, будет следующим: %price%


В форме присутствуют следующие кнопки управления:


ПолеОписание
Add LangСоздает новый словарь с именем, которое было указано в поле “or enter new”.

Edit LangИзменяет название словаря выбранного в выпадающем списке “Select lang” на название указанное в поле “or enter new”.

Delete LangУдаляет словарь, выбранный в выпадающем списке “Select lang”.

Set LangДелает активным словарь, выбранный в выпадающем списке “Select lang”.

Show recordsПозволяет вывести на экран записи словаря, после выставления всех необходимых параметров.

BrowseПозволяет выбрать файл со словарем для загрузки его в выбранный словарь системы системе.

Load dictionary from fileЗагружает в систему словарь из CSV-файла выбранного в поле “Browse”.

Add newДобавляет одну свободную запись в конец словаря, которую можно использовать для новой фразы.

Apply changesСохраняет изменения в содержании выбранной записи словаря.

Добавление и заполнение нового словаря


Для того чтобы добавить и отредактировать новый словарь, нужно выполнить следующие действия:


1. Вписываем название нового словаря в поле “or enter new”.


2. Нажимаем кнопку “Add Lang”. Будет создан новый словарь. По умолчанию все новые словари будут иметь те же самые записи, что и базовый словарь (base).


3. Выбираем любую запись, в правой части экрана есть два поля, расположенные друг над другом. Верхнее поле показывает выбранную фразу, так как она заведена в базовом словаре (эталон). Нижнее поле, позволяет эту же фразу, ее содержимое в текущем словаре. Изменяем содержимое фразы по необходимости в нижнем окне.


4. Сохраняем изменения, нажав кнопку “Apply changes”.


Таким образом, можно перевести все надписи в словаре на другой язык. Чтобы начать использовать новый словарь, нужно его выбрать в выпадающем списке “Select lang” и нажать кнопку “Set Lang”.
