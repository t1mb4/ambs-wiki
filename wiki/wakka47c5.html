Partner pay. Просмотр платежей


Данная форма предназначена для просмотра денежных средств на балансе доступного оператора и, при необходимости, добавления денежных средств на баланс оператора через систему «WebMoney».
Внешний вид формы представлен на следующем рисунке:



Описание полей формы:

ПолеНазначение
Select operatorПозволяет выбрать для просмотра доступного оператора.
Примечание: После выбора для просмотра доступного оператора, появляется табличка с указанием денежных средств на балансе оператора.



В форме присутствуют следующие кнопки управления:

КнопкаНазначение
Add pay. Добавляет денежную сумму на баланс оператора через систему «WebMoney».

Set comment to pay. Открывает список последних проплат по оператору, в котором указано: кто?, сколько? и когда? добавлял на баланс оператора.

Take money.


Примечание: Для использования данного вида проплаты пользователю необходимо наличие установленного WM Keeper Classic версии выше 2.2.0.9
