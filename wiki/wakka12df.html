Пример заведения дилера


Необходимо зарегистрировать дилера с логином «Tester», за которым закреплен оператор «Operator 25», следует сделать следующее:

  • При условии что создана дилерская группа (см. Заведение дилеров в IP Studio AMBS), необходимо открыть форму «Users» в разделе «Administration» для регистрации дилера:
    1. В поле «Enter new or select existing group» из ниспадающего списка выбрать группу «Diller»;
    2. Нажать кнопку «Add new user»;
    3. В открывшейся форме, ввести данные пользователя;
    4. Для подтверждения регистрации нажать кнопку «add user».

См. рис. :



  • открыть форму «Oper rights» в разделе «Administration» и назначить дилеру права на оператора:
    1. Следует выбрать из ниспадающего списка в поле «Select user group» название группы «Diller»;
    2. В поле «Select user» выбрать имя дилера, за которым необходимо закрепить оператора;
    3. Выбрать из списка операторов того оператора, которого необходимо закрепить за дилером поставив галочки на оригинацию и терминацию в полях «View origination» и «View termination»;
    4. Нажать кнопку «Change rights» для подтверждения внесенных параметров.
