Форма “Users”


Раздел для ведения и учета пользователей системы.



Форма состоит из двух зависимых форм:
  • Первая форма для работы с группами пользователей.
  • Вторая (вложенная) форма содержит пользователей выбранной группы.



Изначально присутствует только одна, базовая группа администраторов – Admins. В процессе использования системы можно добавлять новые группы пользователей: Менеджеры, Техники, Аналитики и т.д. Каждой группе пользователей можно назначить свои ограничения и уникальные права доступа к выбранным формам системы.
Пример: группа менеджеров будут иметь доступ только к формам статистики и отчетности, группа техников доступ к формам детализации звонков.

Кнопки управления формы для работы с группами:

КнопкаНазначение
Добавляет новую группу пользователей. Предварительно нужно написать имя новой группы.

Обновляет информацию о существующей группе пользователей (после внесения изменений).

Удаляет выбранную группу.


Форма пользователей состоит из следующих полей:

ПолеОписание
LoginЛогин пользователя. Может содержать только заглавные, прописные буквы латинского алфавита и символ подчеркивания.

User nameФИО пользователя.

All operatorsФлаг доступа к полной информации по операторам, заведенным в системе.
Может принимать следующие значения:
0 – Пользователь имеет выборочный доступ к данным операторов, разрешенных ему для просмотра в форме прав на доступ к данным операторов (Administration – Rights – Operator rights).
1 – Пользователю даны неограниченные права на просмотр данных всех операторов – “All operators” в форме прав.
Значение по умолчанию – 0.

All ratesФлаг доступа к любым тарифным планам операторов, заведенных в системе.
Может принимать следующие значения:
0 – Пользователь имеет выборочный доступ к данным операторов, разрешенных ему для просмотра в форме прав на доступ к тарифным планам операторов (Administration – Rights – Rate rights).
1 – Пользователю даны неограниченные права на просмотр тарифных планов всех операторов – “All rates” в форме прав.
Значение по умолчанию – 0.

All lotsФлаг доступа к карточным лотам, заведенным в системе.
Может принимать следующие значения:
0 – Пользователь имеет выборочный доступ к лотам, разрешенных ему для просмотра в форме прав на доступ к карточным лотам (Administration – Rights – Lot rights).
1 – Пользователю даны неограниченные права на просмотр лотов – “All lots” в форме прав.
Значение по умолчанию – 0.


Работа с формой пользователей аналогична работе с группами:

КнопкаНазначение
Добавляет нового пользователя в группу. После нажатия будет открыто окно для заполнения данных о пользователе.

Открывает окно с данными выбранного пользователя для редактирования.


Окно с данными пользователя:



состоит из следующих полей:

ПолеОписание
User groupНазвание группы, в которую будет добавлен пользователь.

LoginЛогин пользователя, имя с которым пользователь будет входить в систему.
Логин может содержать только заглавные, прописные буквы латинского алфавита и символ подчеркивания.

User name (first, last)ФИО пользователя.

PasswordПароль с которым пользователь будет входить в систему.

Repeat passwordДополнительное поля для ввода пароля пользователя. Контрольное поле.


Примечание: Новый пользователь всегда добавляется без прав на просмотр данных операторов, тарифов или лотов. После добавления нового пользователя нужно обязательно назначить ему права на просмотр информации по операторам, тарифам или лотам.
