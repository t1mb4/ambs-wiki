Форма “Summary Statistic”


Форма позволяет посмотреть общую сводную статистику по трафику за выбранный период времени.



Форма состоит из следующих полей:


ПолеНазначение
Select operatorМожно выбрать конкретного оператора, чтобы получить статистику только по этому оператору. По умолчанию статистика показывается по всем операторам (Any).

Select server MeraВыбор MVTS-сервера, позволяет посмотреть статистику, только по выбранному серверу. По умолчанию, статистика показывается по всем серверам (All).

Direction typeТип трафика в показываемой статистики.
Может принимать следующие значения:
Terminate – Терминация трафика.
Originate – Оригинация трафика.
Roaming – Роуминговый трафик.
All – Показать статистику по всем типам трафика. Каждый тип будет показан в отдельной таблице.
Значение по умолчанию – All.

Stastistic groupТип отображения трафика в статистике.
Может принимать следующие значения:
Operators – Трафик в статистике будет показан с раскладкой по операторам.
Countries – Трафик в статистике будет показан с раскладкой по странам.
Countries-Opers – Трафик в статистике будет показан с раскладкой по странам, с указанием операторов.
Значение по умолчанию – Operators.

Destination groupТип группировки данных в статистике.
Может принимать следующие значения:
By oper dest – По направлениям оригинирующей стороны.
By peer dest – По направлениям терминирующей стороны.
Значение по умолчанию – By oper dest.

Select agentПоле для выбора агента. Можно получить статистику по работе операторов закрепленным за определенным агентом.

Enter first numbers of destinationЕсли нужно посмотреть статистику только по конкретному направлению, нужно в это поле ввести первые цифры кода этого направления.

Time periodПериод времени, за который будут показаны звонки.
Может принимать следующие значения:
Today – Сегодня.
Yesterday – Вчера.
3 days – Три дня.
7 days – Семь дней, не включая текущий день.
Month – Текущий месяц.
Last month – Предыдущий месяц.
Period – Период, заданный вручную полями From-To.
Last 24 hours – Период за последние 24 часа, указывается в часах
Значение по умолчанию – today

Period From-ToДаты начала и завершения расчетного периода, за который будет выбираться статистика (задаются вручную, с помощью календарей).


В форме присутствуют следующие кнопки управления:


КнопкаОписание
Show statВыводит на экран сводную статистику по работе операторов в форме таблицы.

Show SVOD parametersВыводит на экран окно с параметрами полей, показываемых в таблице статистики.
Лишние поля можно отключить и не показывать в таблице статистики.

Show graphЗапускает процесс вывода на экран сводной статистики по работе операторов в виде графика.

Show summary trafficВыводит на экран отчет по прошедшему объему трафика в форме таблицы.
В данном виде отчета будет показан объем трафика за указанный период времени с шагом в сутки или в час.


При нажатии на кнопку “Show SVOD parameters” будет открыто окно с параметрами полей статистики:



Визуально окно можно разделить на две части. В левой части настраиваются поля сводной статистики вызываемой кнопкой “Show stat”.
В правой части расположены параметры относящиеся с статистике по объему трафика вызываемой кнопкой “Show summary traffic”.
Статистику по объему трафика можно автоматически отправить на выбранный электронный почтовый ящик.


Для того чтобы просмотреть сводную статистику нужно нажать на кнопку “Show stat”.



Таблица статистики состоит из следующих столбцов:


ПолеНазначение
OperatorНазвание оператора.

All minutesОбщее количество минут прошедшее через шлюзы оператора.

Our profitЧистая прибыль с трафика по данному оператору.

All callsОбщее количество звонков.

Success callsКоличество успешных звонков, у которых коды возврата -16,17 и 19.

Short callsКоличество коротких звонков, у которых продолжительность меньше 30 секунд.

Origination costСтоимость оригинации трафика.

Termination costСтоимость терминации трафика.

ASRПроцент успешных звонков.

Complete callsКоличество состоявшихся звонков, у которых продолжительность больше нуля.

ACDСредняя продолжительность разговора по звонкам оператора.

PercentПроцент прибыли, которую составляет прибыль выбранного оператора от общей прибыли.

AgentАгент, который нашел донного оператора.

Termination agent profitСумма прибыли терминационного агента.

Origination agent profitСумма прибыли оригинационного агента.


Для того чтобы просмотреть статику в виде графика нужно выбрать шаг графика:


ПолеНазначение
Graph typeШаг в графике. Бывает двух типов:
By day – Строить график с интервалом в один день.
By hour – Строить график с интервалом в один час.


И выбрать тип графика:


ГрафикОписание
Calls countПостроить график по количеству звонков прошедших за единицу времени.

MinutesПостроить график по количеству минут.

ProfitПостроить график прибыли за выбранный период. Данную диаграмму можно строить как по всем операторам, так и индивидуально по отдельному оператору.



Чтобы посмотреть статистику по объему трафика в целом за период времени, нужно воспользоваться кнопкой “Show summary traffic”.
