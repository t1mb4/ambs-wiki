Пример создания и работы с тарифным планом


Рассмотрим на примерах, как создавать, удалять тарифные планы и добавлять в них новые направления.


Посмотреть пример добавления тарифного плана


Пример 1. Создание нового тарифного плана


Для создания нового тарифного плана:



  1. Вписываем имя нового тарифного плана в поле “Rate name”.
  2. Нажимаем кнопку , чтобы добавить тарифный план.

Пример 2. Изменение имени тарифного плана


Для изменения имени тарифного плана:



  1. Выбираем тарифный план, который будем переименовывать.
  2. Вписываем новое имя для тарифного плана в поле “Rate name”.
  3. Нажимаем кнопку , чтобы сохранить внесенные изменения.

Пример 3. Удаление тарифного плана


Для удаления тарифного плана:



  1. Выбираем тарифный план, который будем удалять.
  2. Нажимаем кнопку , чтобы удалить тарифный план.

Примечание: Тарифный план можно удалить, только если он не закреплен за сайтами операторов. Если он закреплен, предварительно этот тарифный план нужно отключить в разделе операторов, чтобы он не использовался и после этого удалять.


Пример 4. Импорт направлений из CSV-файла


Для импорта направлений из CSV-файла в тарифный план нужно:



  1. Выбираем тарифный план, куда будем импортировать направления из файла.
  2. С помощью кнопки «Обзор...» выбираем CSV-файл, который будем загружать.
  3. Нажимаем кнопку , чтобы загрузить направления из CSV-файла в тарифный план.

Примечание: После процесса импорта, рядом с кнопкой появится ссылка “Show load rate LOG” на лог загрузки: если в процессе импорта были ошибки, незагруженные строки будут записаны в этом логе с примечаниями, почему они не были загружены. Рекомендуем просматривать лог после каждой операции импорта, чтобы быть уверенными, что все направления загружены правильно.


Пример 5. Добавление единичного направления в тарифный план


Для добавления одного выбранного направления в тарифный план нужно:



  1. Выбрать тарифный план, куда будем добавлять направление.
  2. В поле фильтра ввести код направления, которое будем добавлять (в нашем примере это “7495”) и с помощью кнопки “Filter” найти все направления, в которых встречается искомый код.
  3. В окне с имеющимися в системе направлениями, содержащими код “7495”, выбрать строку с нужным набором кодов.
  4. Нажать кнопку “Add destinations”, чтобы добавить выбранное направление в тарифный план.

Примечание: Новое направление будет добавлено с ценой “0.00” и иметь статус “Closed”. После операции добавления, новое направление нужно разблокировать в тарифном плане для редактирования и изменить его цену и статус, чтобы оно стало рабочим.
