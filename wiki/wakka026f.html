Форма “Calls Online”


Форма позволяет посмотреть звонки, проходящие в текущий момент времени на MVTS-сервере (аналог команды “show calls” mvts-сервера):



Статистика звонков динамически обновляется через указанный, в поле “Refresh time”, промежуток времени и таким образом позволяет просматривать статистику звонков в реальном времени. Если параметры отображения звонков изменены, эти изменения вступят в силу после нажатия на кнопку “Save options” или “Set parameters”, соответственно. Вы можете задать следующие параметры:

ПолеНазначение
Select originate/terminate operatorВыбрать конкретного оригинирующего или терминирующего оператора для просмотра только его звонков.

Select originate/terminate ipЕсли вы выбрали конкретного оператора, в этом поле можно выбрать конкретный ip-адрес оператора для просмотра.

Select mera serverПоле для выбора определенного MVTS-сервера, где вы хотите просмотреть звонки.

Enter destination numberЕсли требуется получить лист звонков, начинающихся на определенные цифры кода, то в этом поле нужно ввести первые цифры.

Refresh time (sec.)Время обновления информации, как часто AMBS будет забирать данные с MVTS-сервера и обновлять страницу. Не рекомендуется ставить это значение меньше 10 секунд.

Destination numberНомер вызываемого абонента.

HostХост-сервер, через который идут звонки.

Src bytes inКоличество байт, принятых от оригинирующего оператора

Src bytes outКоличество байт, отправленных оригинирующему оператору.

Destination ipПоказывать адрес получателя.

Originate operatorПоказать названия оригинирующих операторов.

Source ipПоказывать адрес, откуда пришел звонок.

Elapsed timeВремя, прошедшее с начала звонка.

Dst bytes inКоличество байт, принятых от терминирующего оператора.

Terminate operatorПоказать название терминирующих операторов.

Dst bytes outКоличество байт, отправленных терминирующему оператору.

Source NumberНомер вызывающего абонента.


Есть несколько вариантов просмотра текущих звонков:

details
Будет показан полный список всех звонков, проходящих через софтсвич с детальной информацией по каждому звонку.




summary
Суммарная статистика по звонкам, проходящим через хост, количество нулевых и не нулевых звонков .





summary operator
Суммарная статистика звонков по операторам. Сколько сейчас проходит звонков от и на каждого оператора.



Примечания:
  1. Не рекомендуется использовать период обновления данных меньше 10 секунд, при детальным режиме отображения информации это может привести к проблемам в работе софтсвича.
  2. При прохождении звонка через шлюз заданный в АМБС как «клон» или «элемент подсети», название сайта такого шлюза в статистике видно не будет.
Ассоциация таких типов шлюзов с конкретными сайтами происходит на более позднем этапе.
