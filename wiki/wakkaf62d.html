Форма “Dealer Sales”


Форма для расчетов с клиентами по карточкам. В этой форме можно вести учет финансовых операций и расхода карточек.



Для просмотра истории операций с карточками, необходимо задать период времени, за который будет показаны движения карточек, выбрать тип отчета: суммарный или детальный и при необходимости установить конкретные параметры отчета:

ПолеНазначение
Select dealerКонкретный дилер карточек, по которому будет показан отчет.
Значение по умолчанию: Any (по всем дилерам)

Select LotКарточный лот, по которому будет показан отчет.

Select point countОтчет по карточкам указанного номинала.


Кнопки управления:

КнопкаНазначение
Show infoПросмотреть статистику по операциям с карточками.

Sale cardДобавить новую запись с продажей карточек, после заполнения всех деталей продажи.

Card returnОформить возврат карты от клиента, после окончания заполнения всех деталей возврата.

Delete saleУдалить выбранную операцию из базы, перед удалением необходимо активировать запись, поставив галочку в первое поле записи.


Для того чтобы произвести продажу или возврат карт нужно:
  1. Выбрать дилера в поле “Dealer”.
  2. Выбрать лот в поле “Lot prefix”.
  3. Указать стартовый номер (с какого номера были проданы карточки) в поле “Serial start”.
  4. Задать номер последней проданной карты из этого лота в поле “Serial stop”. Количество карт будет рассчитано автоматически.
  5. Выставить номинал продаваемых карт в поле “Points count”.
  6. Цену за единицу номинала в поле “Price”. Суммарная цена будет рассчитана автоматически в зависимости от количества карт.
  7. Дату продажи в поле “Sale date” по умолчанию ставится текущая дата.

Далее, в зависимости от операции, нужно нажать кнопку “Sale cards” для внесения в базу операции продажи карт или кнопку “Card return” для регистрации возврата карт.
