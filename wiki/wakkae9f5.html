Форма “Rights”


Подраздел для настройки прав пользователей на объекты системы, состоит из следующих форм:
  • Форма для настройки прав доступа групп к формам системы
  • Форма прав на просмотр данных операторов
  • Форма для просмотра тарифных планов операторов
  • Форма для просмотра карточных лотов

Права на доступ к формам системы назначаются группе пользователей.
Права на доступ к данным операторов их тарифным планам и лотам назначаются каждому пользователю отдельно.

Изначально (после создания), у группы нет прав доступа к формам и у пользователей нем прав доступа к данным операторов их нужно назначать отдельно.

Для некоторых типов пользователей в биллинге есть предустановленные группы:
  1. Для администраторов системы, с полным доступом ко всем формам – группа “Admins”
  2. Для карточных пользователей, только с доступом в форму статистики по карте – группа “Cards”
  3. Для партнеров(операторов) которые будут только просматривать статистику по своей работе – группа “Partners”
  4. Для диллеров (операторы с возможностью маршрутизации своего трафика) – группа Dialers
