Форма “Technical Statistic”


Здесь можно просмотреть техническую информацию по прошедшим звонкам операторов, за любые промежутки времени.



Форма раздела состоит из следующих параметров:

ПолеНазначение
Select originate operatorПозволяет выбрать для просмотра конкретного оператора, от которого пришли звонки.
Значение по умолчанию: Any (любые операторы).

Select terminate operatorПозволяет выбрать для просмотра конкретного оператора, на которого ушли звонки.
Значение по умолчанию: Any (любые операторы).

Time PeriodПериод времени, за который будут показаны звонки.
Может принимать следующие значения:
Today – Сегодня.
Yesterday – Вчера.
3 days – Три дня.
7 days – Семь дней, не включая текущий день.
Month – Текущий месяц.
Last month – Предыдущий месяц.
Period – Период, заданный вручную полями From-To.
Значение по умолчанию – today

Select time fromВремя начала и конца выборки, будут анализироваться только звонки прошедшие в указанный период времени.
Значение по умолчанию: 00:00-24:00

Time differenceСдвиг часового пояса в часах. Используется для сдвига прошедших звонков при сверке с операторами.
Значение по умолчанию: 0 часов.

Destination codeФильтр начальных цифр номера. Позволяет получить статистику по звонкам прошедшим на определенное направление.

Show typeТип требуемой статистики.
Q931 – Статистика по кодам завершения
Src codec – Статистика по кодекам оригинации
Dst codec – Статистика по кодекам терминации
Elapsed time- Статистика показателей качества

Mera serverЕсли используется несколько MVTS-серверов, можно получить данные по конкретному серверу, выбрав его в этом списке.
