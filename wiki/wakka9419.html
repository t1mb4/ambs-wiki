Форма “LCR Reports”


Форма позволяет получить на указанный почтовый ящик однодневный отчет по терминирующим операторам их трафику и ценам на направления.



Описание полей формы:

ПолеНазначение
Report dateДень, за который будет строиться отчет.

Originate operatorОригинирующий оператор для которого строится отчет. В отчет войдут только те терминирующие направления, на которые может пройти трафик этого оператора, с учетом его цен и правил маршрутизации.

Terminator limitКоличество терминирующих направлений для каждого направления оригинатора.

CountryСтрана. Можно указать конкретную страну, для которой будет строиться отчет.

Mail toПоля для выбора электронного почтового адреса куда будет отправлен отчет.
