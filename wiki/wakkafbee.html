Форма “Time Zones”


Форма для работы с временными периодами. Далее временные периоды можно будет добавлять в тарифные планы, таким образом делая их более гибкими. Например, можно будет настроить работу одного и того же направления в различные периоды суток по разной цене или создать «праздничный» или «выходной» тарифные планы. Внешний вид формы показан на рисунке:



Форма раздела состоит из шапки с описанием полей и листа временных периодов, заведенных в системе. Описание полей формы:

ПолеНазначение
Time zone nameНазвание временных зон, заведенных в системе.

Week scheduleНедельное расписание работы. Периоды работы временных зон в течение недели.

Included datesВключенные даты работы. Список дат в году, когда зона будет активна.

Included days periodsПериоды активности в течение дат работы.

Excluded daysИсключенные даты. В эти даты данная временная зона будет неактивна.


В листе временных периодов присутствуют следующие кнопки управления:

КнопкаНазначение
Добавляет новую пустую временную зону, и открывает форму для редактирования ее параметров.

Вызывает форму для редактирования параметров выбранной временной зоны.

Копирует выбранную временную зону, создает копию временной зоны.


Если нажать на кнопку , будет открыто окно с параметрами выбранного временного периода:



Каждый временной период может состоять из следующих параметров:

ПолеНазначение
NameНазвание временной зоны. При необходимости, название может быть изменено.

Week scheduleДобавленные периоды активности зоны в течение недели.

Included datesВключенные дни работы. Список дней в году, когда зона активна.

Included days periodsПериоды активности в течение включенных дней.

Excluded datesИсключенные дни. Дни, когда данная временная зона будет не активна.

Add period to week scheduleПоля для добавления периодов активности зоны в течение недели.

Add included dayПоля для добавления нового включенного в работу дня.

Add included days periodПоля для добавления периодов активности для включенных дней.

Add excluded dayПоля для добавления нового исключенного из работы дня.

И содержать следующие кнопки управления:

КнопкаНазначение
Добавляет в параметры временной зоны новый установленный период времени, в недельное расписание или для включенных дней соответственно.

Добавляет в параметры временной зоны новый включенный день.

Добавляет в параметры временной зоны новый исключенный день.

Удаляет из заданных параметров временной зоны выбранные периоды активности и даты. За один раз можно выбрать несколько периодов и дат, для этого нужно нажать и держать нажатой во время выбора клавишу “Ctrl”.

Полностью удаляет текущую временную зону.

Сохраняет внесенные изменения и закрывает окно с параметрами временной зоны.

Закрывает окно с параметрами временной зоны без сохранения внесенных изменений.


Для добавления нового временного периода достаточно нажать на кнопку и заполнить параметры временного периода.
