Примеры работ с картами

1. Пример создания группы карточек.

В созданном лоте (см. «Пример работ с лотами») сгенерируем группу карточек со следующими параметрами:


  • количество карт – 10;
  • авторизация карты – по пину;
  • стоимость карты – 5 единиц;
  • первая карта группы должна иметь номер – 100;
  • пины создаваемых карт должны начинаться с цифр – 311;
  • количество одновременных звонков – 1;
  • категория создаваемых карт – debit;

1. Следует открыть форму по созданию карточек, путем нажатия кнопки «Generate cards» в лоте;
2. В поле «Cards generate» выставить режим «Group»;
3. В поле «Authorization type» выставить тип – PIN;
4. В поле «Amount» выставить количество единиц – 5;
5. В поле «Cards count» выставить количество карточек – 10;
6. В поле «Pin prefix» выставить – 311;
7. В поле «Max. sessios» – 1;
8. В поле «Serial start» – 100;
9. В поле «Category» – debit;
10. Нажать кнопку «Apply change».




2. Пример блокирования одной карточки.

Для того что бы заблокировать карточку, необходимо выполнить следующее:


1. Поставить галочку в пустое поле слева от поля «Card number» для той карточки, которую необходимо заблокировать;
2. В поле «Change status» выбрать значение «Selected cards» и поставить параметр «Blocked»;
3. Нажать кнопку «Change status» для подтверждения блокировки карточки.



3. Пример заведения личной карточки.

Что бы присвоить карточку определенному абоненту, необходимо выполнить следующее:


1. В поле «Card number» кликнуть на ссылку названия карточки – vt101;
2. Откроется форма «Card statistic», где следует нажать кнопку «Set personal card»;
3. В открывшейся форме ввести ФИО владельца карточки;
4. Нажать кнопку «Set card» для подтверждения внесенных параметров.




4. Пример поиска карт по выбранному диапазону.

Если необходимо найти группу карточек в составе лота – это можно осуществить следующим образом:


1. В поле «Filter» выбрать значение «Card number range»;
2. По выбранному значению «Card number range» поставить параметр «Range»;
3. Ввести названия карт (начальное и конечное) для диапазона карт, которые необходимо найти;
4. Нажать кнопку «Find» для осуществления поиска по указанному диапазону.



5. Пример поиска карты по имени владельца карточки.

Если необходимо найти карточку в составе лота по имени абонента – это можно осуществить следующим образом:


1. В поле «Filter» выбрать значение «First name»;
2. Ввести в пустое поле справа имя абонента (Иванов), карточку которого необходимо найти;
4. Нажать кнопку «Find» для осуществления поиска по указанному значению.



6. Пример смены пин-ключа абонентом в форме «User interface».

Для того что бы сменить пин-ключ, абоненту необходимо сделать следующее:


1. В поле «Card number» ввести название карточки – vt101;
2. В поле «PIN code» ввести пин-код и пин-ключ карточки – 311821 и 144807;
3. Нажать кнопку «Find» для поиска карточки по введенным параметрам;
4. В поле «change pin-key» сменить пин-ключ (сообщение «must be “_” numbers» указывает из какого количества цифр должен состоять пин-ключ);
5. Нажать кнопку «Change pin key» для подтверждения смены пин-ключа.
