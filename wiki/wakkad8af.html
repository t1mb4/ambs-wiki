Форма “Operators Reports”. Отчет по операторам.


Форма позволяет просмотреть список всех операторов заведенных в системе, адреса, закрепленные за ними тарифы и краткую статистику по их объемам трафика за указанный период времени. Также здесь можно найти нужного оператора по его контрольным параметрам: адрес шлюза, часть почтового адреса, номер icq и т.д.



Для просмотра отчета по всем операторам в системе, достаточно нажать кнопку поиска ().
Для того чтобы начать процесс поиска конкретного оператора, нужно задать хотя бы один из параметров оператора и после этого нажать на кнопку поиска. Результат будет представлен в виде сводной таблицы с учетом введенных параметров.



Поиск оператора можно производить по его ip-адресу, номеру факса, ICQ-номеру или почтовому адресу (по полному адресу или по доменному имени, например так: @mail.ru ). Для этого достаточно вписать данные оператора в соответствующее поле поиска и нажать кнопку поиска ().


Следует учитывать, что в режиме просмотра только «рабочих» шлюзов будут показывать только те шлюзы, по которым уже прошел трафик.
Новые, только что добавленные шлюзы можно увидеть только в режиме «все» или «не активные».


В полученном списке с результатами, названия операторов, тарифы операторов и адреса контактов, при нажатии на них, позволяют перейти в раздел с параметрами выбранного оператора, просмотреть тариф оператора или написать письмо выбранному контакту, соответственно.
