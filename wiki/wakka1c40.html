Пример добавления нового шлюза


Чтобы правильно добавить новый ip-адрес на сайт оператора, необходимо выполнить следующие действия:


1. В разделе “Edit Operators”, выбираем оператора, которому нужно добавить шлюз.


2. Выбираем сайт оператора, куда будем добавлять новый шлюз.


3. В секции шлюзов, нажимаем кнопку для добавления нового шлюза.


4. Заполняем обязательные поля параметров шлюза.


5. Нажимаем кнопку , чтобы добавить новый шлюз.


Замечания:


1. Каждый новый шлюз должен иметь свой уникальный идентификатор (Gateway ID).


2. Если новый шлюз не добавился, внимательно прочитайте сообщение, которое выдает система при попытке добавить этот шлюз.
Возможно в системе уже имеется аналогичный шлюз, в этом случае нужно добавлять шлюз с использованием правила «использования одного ip-адреса несколько раз».


3. Если при добавлении шлюза вам неизвестны некоторые из параметров работы шлюза, вы можете оставить параметры предлагаемые по умолчанию (или сделать шлюз неактивным), и добавить его в систему. Впоследствии, после уточнения всех дополнительных параметров шлюза их можно будет добавить через режим редактирования шлюза.
