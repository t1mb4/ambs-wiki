Секция АОН-групп оператора


Секция АОН-групп вызывается путем нажатия кнопки «Edit ANI» в секции параметров оператора.


АОН (ANI) – автоматический определитель номера, правила набора номера.
АОН-группа – совокупность АОН-сетов.
АОН-сет – совокупность АОН-номеров.

Примечание: идентичные АОН-номера могут входить в состав различных АОН-сетов.



Описание полей формы:

ПолеНазначение
Group Позволяет выбрать для просмотра конкретную группу с АОН-номерами.


Set of numbers Позволяет выбрать для просмотра конкретный сет (набор) АОН-номеров в составе группы. Поле работает в связке с полем Group.

Add ANI prefix Позволяет добавить правило набора номера, который представляет из себя последовательность цифр. Правила такие же как и к набираемому номеру.

Import ANI list Позволяет импортировать АОН-номерами в файл формата .csv.

To: Позволяет переместить АОН-номера в другой сет в составе группы. Становится активным если значение кнопки «Move selected»: true.
(Перенос АОН-номеров в другой сет осуществляется путем выделения АОН-номеров, необходимых для переноса в другой сет – далее активировать кнопку «Move selected» – выбрать в поле «To» необходимый сет – нажать кнопку «Move selected ANI»).



В форме присутствуют следующие функциональные кнопки:

КнопкаНазначение
Add group. Добавляет новую АОН-группу.

Delete group. Удаляет выбранную АОН-группу.

Add set. Добавляет новый АОН-сет.

Delete set. Удаляет выбранный АОН-сет.

Delete ANI-Exp. Удаляет, отмеченные галочкой, АОН-номера из выбранного сета.

Add ANI. Добавляет новый АОН-номер.

Примечание: в поле Add ANI prefix поддерживаются все стандартные форматы regexp-ов (регулярных выражений, см. приложения) для заведения групп номеров.



Move selected. Активизирует поле To: для переноса АОН-номеров в другой АОН-сет. Может принимать два значения: True и False.
Значение по умолчанию:False.

Move selected ANI. Осуществляет перемещение АОН-номеров в выбранный из ниспадающего списка в поле «to:», АОН-сет.

Open. Позволяет указать путь к файлу АОН-номеров для импортирования.

Import ANI. Позволяет загрузить файл с АОН-номерами в программу.

Export ANI list. Позволяет экспортировать в файл АОН-номера оператора.



Примечание:
1. АОН-номера после добавления не редактируются, только удаляются и добавляются по новой.
2. Если необходимо продублировать или перенести список АОН-номеров из одной группы в другую, следует воспользоваться функцией экспорта (Export ANI list), а затем функцией импорта (Import ANI list).


Примеры работы с формой:


1. Заведение АОН-группы:
  • следует ввести в пустое поле (1), слева от кнопки «Add group», название АОН-группы;
  • нажать кнопку «Add group» (2).
См. рис.:



2. Заведение АОН-сета:
  • следует в поле «Group» (1) выбрать название АОН-группы, в которой необходимо завести АОН-сет;
  • далее в пустое поле (2), слева от кнопки «Add set», ввести название АОН-сета;
  • нажать кнопку «Add set» (3).
См. рис.:



3. Заведение АОН-номера:
  • следует в поле «Group» (1) выбрать название АОН-группы;
  • далее в поле «Set of numbers» (2) выбрать название АОН-сета;
  • затем в поле «Add ANI prefix» (3) ввести АОН-номер;
  • нажать кнопку «Add ANI» (4).
См. рис.:



Примечание: Отчеты по АОН-номерам следует смотреть в разделе Reports – форма «ANI reports».


4. Перенос АОН-номера (группы АОН-номеров) в состав другого АОН-сета через функцию переноса:
  • следует выделить АОН-номер (группу АОН-номеров) галочкой (ми) (1);
  • далее активировать кнопкой «Move selected» (2) поле «To:» (3) и выбрать в ниспадающем списке АОН-сет, в который необходимо перенести АОН-номера;
  • следует нажать кнопку «Move selected ANI» (4) для завершения процесса переноса АОН-номеров.
См. рис.:



5. Перенос АОН-номеров из АОН-сета в состав АОН-сета другой АОН-группы через функцию экспорта:
  • следует выбрать в АОН-группе (1) необходимый АОН-сет (2);
  • нажать кнопку «Export ANI list»(3), информация сохранится в файл;
  • выбрать АОН-группу(4), в которую необходимо загрузить АОН-номера и АОН-сет(5), в который непосредственно следует загрузить АОН-номера;
  • в поле «Import ANI list» (6)указать путь к файлу, воспользовавшись кнопкой «Open» (7);
  • нажать кнопку «Import ANI» (8) для загрузки из файла АОН-номеров.
См. рис.:





6. Удаление АОН-номера:
  • следует в поле «Group» (1) выбрать название АОН-группы;
  • далее в поле «Set of numbers» (2) выбрать название АОН-сета;
  • затем в появившемся списке АОН-номеров выделить галочкой удаляемый АОН-номер (3);
  • нажать кнопку «Delete ANI-Exp» (4).
См. рис.:



7. Удаление АОН-сета:
  • следует в поле «Group» (1) выбрать название АОН-группы;
  • далее в поле «Set of numbers» (2) выбрать название удаляемого АОН-сета;
  • нажать кнопку «Delete set»(3).
См. рис.:



8. Удаление АОН-группы:
  • следует в поле «Group»(1) выбрать название АОН-группы;
  • нажать кнопку «Delete group» (2).
