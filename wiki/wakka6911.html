Конвертирование CSV-файлов с направлениями


В процессе работы постоянно возникают ситуации, когда Excel-файл с направлениями имеет большой объем и преобразовывать его вручную к виду системы займет довольно продолжительный период времени. Для таких ситуаций в системе есть функция автоматического конвертирования CSV-файлов из общего формата записи в формат записи системы.


В процессе конвертирования над записями направлений будут произведены следующие действия:


  • расположение полей в направлениях будет приведено к виду системы,

  • полный код направления, записанный в виде одной записи, будет разделен на два поля в соответствии с правилами системы. Разделение будет производиться на основании имеющихся кодов стран в базе системы,

  • направления, у которых совпадают первые цифры кодов направления, и их цена будут объединены в одну строку и записаны с учетом регулярных выражений системы.

Чтобы воспользоваться системой конвертирования, достаточно привести Excel-файл с направлениями к следующему виду:



Результат нужно сохранить в отдельный CSV-файл в формате – *.CSV (MS-DOS, разделитель точка с запятой).
Далее нужно в форме “Edit tariffs” (Tariffs – Edit tariffs) выбрать этот файл с помощью кнопки “Browse”(Обзор) и нажать кнопку .
По завершении процесса конвертирования, рядом с кнопкой будет показана ссылка “waiting form”, на страничку для результатов.
Перейдя по ссылке на страничку результатов, полученный файл можно будет сохранить на локальный ПК, нажав на ссылку “converted file” правой кнопкой мыши и выбрав пункт «Сохранить объект как...».


Результат в виде CSV-файла, можно открыть с помощью программы «Блокнот» и проверить на наличие ошибок и неточностей.
