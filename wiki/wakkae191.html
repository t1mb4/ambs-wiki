Форма “Invoices”


Форма для создания счетов операторам за выбранный расчетный период. Позволяет подготовить счет (далее инвойс) по:
  • выбранному оператору,
  • карточному лоту
  • по конкретной карте.

Внешний вид формы инвойсов показан на рисунке:



Описание полей формы:

ПолеНазначение
Select operator / Card number / Select LotПозволяет задать для какого объекта нужно построить инвойс: оператор, лот, карта.

Terminate / Originate / AllЗадается тип трафика для которого будет строиться инвойс: оригинация, терминация, все

Time periodПериод времени, за который будут показаны звонки.
Может принимать следующие значения:
Today – Сегодня.
Yesterday – Вчера.
3 days – Три дня.
7 days – Семь дней, не включая текущий день.
Month – Текущий месяц.
Last month – Предыдущий месяц.
Period – Период, заданный вручную полями From-To.
last 24 hours – Период за последние 24 часа, указывается в часах
Значение по умолчанию – today

Period from-toДаты начала и завершения расчетного периода, за который будет выбираться статистика (задаются вручную, с помощью календарей).

First numbers of destinationПервые цифры кода направления. Позволяет построить инвойс только по указанному направлению.

Show round minutesВ инвойсе будут показываться округленные минуты, в соответствии с условием округления заданного в тарифном плане.

Full CodesПоказывать в инвойсе полную строку кодов направления или только первые 10 кодов.
например, направления состоят из большого числа кодов, вывод всех кодов может занять очень много места и сделает счет не читабельным (особенно для тарифных планов типа A-Z).
С помощью этой функции можно получить короткие записи диапазонов кодов.

Group byГруппирует направления одинаковые по цене и названию.


В форме присутствуют следующие кнопки управления:

КнопкаНазначение
Generate fast invoiceЗапускает процесс построения инвойса.
Результат будет показан в виде html-страницы в новом окне браузера.

Generate invoice to ExcelЗапускает процесс построения инвойса.
Результат будет показан в виде xls-страницы.

PDFЗапускает процесс создания инвойса.
Результат будет доступен в виде pdf-страницы.


Примечание: Для того, чтобы система могла формировать счета в формате PDF, на сервере должен быть установлен пакет “htmldoc_1.8.23.orig.tar.gz”.
Пакет можно взять с нашего сервера обновлений командой: 'scp billinst@213.219.246.22:htmldoc_1.8.23.orig.tar.gz ./'


Для того чтобы создать новый инвойс необходимо выполнить следующие действия:

1. Выбрать оператора, карточный лот или карту, которому будет выставляться счет в выпадающем списке “Select operator”.

2. Выбрать тип трафика, который будет включен в инвойс: оригинирующий, терминирующий, роуминговый или все типы вместе.

3. Выбрать расчетный период времени.

4. Запустить процесс создания инвойса, нажать на кнопку “Generate fast invoice”.

В новом окне браузера будет показан инвойс.



Для того чтобы в шапке инвойса показывалась картинка с логотипом компании, ее нужно загрузить в систему в форме “Company”.
Картинка должна иметь размер: 780х200 .
